## Description

Use when the request is clearly for instrumental music, soundtrack, score, ambient, background music, game music, film cue, meditation, or study music.

## Instructions

- Treat the request as non-vocal unless the user explicitly asks for a choir or voice texture.
- Convert the request into a clean MiniMax prompt centered on scene, mood, instrumentation, pacing, and emotional arc.
- Prefer concrete instrumentation and arrangement language over metaphor alone.
- Make the prompt describe progression: opening texture, build, climax, landing.
- Keep evaluation texts focused on audible properties such as instrumentation, atmosphere, and motion.
- Avoid unnecessary lyrical framing.
