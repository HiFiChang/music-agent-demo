## Description

Use when the request is mainly for beats, loops, producer-oriented tracks, hip-hop instrumentals, electronic grooves, club ideas, or rhythm-first generation.

## Instructions

- Make rhythm, groove, tempo feel, low-end character, and drum texture the primary control surface.
- Use concrete beat-production language: bounce, swing, drum pocket, bass texture, synth stab, pad bed.
- Keep prompts short and literal. Avoid cinematic over-description unless the user explicitly asks for it.
- Evaluation texts should emphasize beat identity, energy, and arrangement clarity.
