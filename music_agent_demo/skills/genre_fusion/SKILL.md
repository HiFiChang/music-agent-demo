## Description

Use when the request explicitly mixes two or more genres or asks for an unusual cross-genre blend that benefits from deliberate balancing.

## Instructions

- Turn the request into a prompt that names the fusion clearly but limits itself to the 2-3 most important genre anchors.
- Describe what each genre contributes: rhythm source, harmonic color, instrumentation, vocal style, or production polish.
- Remove conflicting excess descriptors. The result should feel intentional, not like a bag of tags.
- Evaluation texts should separately capture the main genre anchors and the fused outcome.
