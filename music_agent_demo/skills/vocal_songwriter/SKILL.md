## Description

Use when the request is for a complete song with vocals, singer presence, hook, chorus energy, lyrical feeling, or a clear vocal pop/R&B/rock/folk structure.

## Instructions

- Build the prompt around vocal identity first, then genre, mood, instrumentation, and production.
- If the user did not provide fixed lyrics, keep the loop prompt-centric and let MiniMax auto-generate lyrics.
- Translate vague requests like “more emotional” into audible details: intimate vocal, airy harmonies, bigger chorus lift, warmer bass, tighter drums.
- Keep the prompt concise and generator-friendly.
- Make evaluation texts capture genre, vocality, emotional core, and production texture.
